# SQL Practice — Joins, Subqueries and CASE

**Tool:** SQL Server · **Built:** June 2026

[← Back to home](../index.md)

## What this is

A worked set of 31 SQL questions covering the core of analytical querying: joins
(inner, left and the rest), subqueries, and CASE logic for banding and recoding.
The query files and a reference cheat sheet are kept together so the working is
visible, not just the answers.

This is the grounding under the dashboards. The 1/0-flag-to-rate move in the HR
Attrition piece is the spreadsheet version of `AVG(flag)`. The measures in the
Power BI builds are the same aggregation logic in DAX. SQL is where that thinking
was drilled first.

## Coverage

- Joins across related tables, including left joins and the cases where rows go
  missing if you pick the wrong one.
- Subqueries for filtering and derived values.
- CASE for banding continuous values and recoding categories.
- A cheat sheet kept as a quick reference.
