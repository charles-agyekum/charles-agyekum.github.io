## How it is built

Plain Markdown rendered by Jekyll (`jekyll-theme-cayman`). The published homepage
is `index.md`. Each project has its own page under `projects/`.

## Running it locally (optional)

You do not need to. GitHub builds the site on push. If you ever want a local
preview you would install Ruby and Jekyll, then `bundle exec jekyll serve`.

## Method

Clean first and document it. Verify every headline figure against the source
before calling a piece done. Close on a recommendation, not a description.

See `SETUP.md` for the publish steps.
